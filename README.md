# Shah Sawar Portfolio

This is my personal portfolio website built with **Next.js**, **React**, and **Tailwind CSS**.

## Features

- Modern, fast, and responsive design
- Showcases projects, skills, and experience
- Smooth scrolling and interactive UI components

## Tech Stack

- Next.js (App Router)
- React
- Tailwind CSS
- Google Fonts (Geist)
- Vercel deployment

## Deployment

The site is deployed on Vercel. You can view it live [here](https://your-portfolio-url.com).
